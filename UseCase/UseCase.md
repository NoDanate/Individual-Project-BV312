Общее описание
Приложение предназначено для прослушивания аудиокниг на мобильных устройствах с системой android

Акторы
1. Администратор
   Прецеденты:
   -Добавить книги - добавление в базу данных новых аудиокниг
   -Удалить книги - удаление из базы данных аудиокниг
   -Редактировать карточки книг - изменение информации об аудиокнигах (такие как название, автор, описание,
    изоображение и др.)

2. Пользователь
   Пецеденты:
   -Создать аккаунт - регистрация в системе для использования приложения
   -Найти аудиокнигу - найти нужую аудиокнигу в каталоге по названию или автору
   -Слушать аудиокнигу - начать прослушивание выбранной аудиокниги
   -Загрузить аудиокнигу для прослушивания - загрузить аудиокнигу из каталога для офлайн прослушивания, либо
    загрузить в приложение свой файл из устройства
   -Посмотреть список рекомендованных книг - рекомендации к книге по выбранной в данный момент книге по жанру
   -Оставить оценку на прослушанную книгу - поставить рейтинг (от 1 до 5) к прослушанной аудиокниге
   Дополнительные прецеденты:
   -Добавить книгу в раздел "понравившиеся" - добавить книгу в закладки для последующего прослушиваня
   -Выбрать диктора - выбрать голос диктора, который будет читать данную книгу
