# Описание приложения

Приложение предназначено для прослушивания аудиокниг на мобильных устройствах с системой Android.

---

## Акторы

### Администратор

**Прецеденты:**

- **Добавить книги** — добавление в базу данных новых аудиокниг.
- **Удалить книги** — удаление из базы данных аудиокниг.
- **Редактировать карточки книг** — изменение информации об аудиокнигах (название, автор, описание, изображение и др.).

---

### Пользователь

**Прецеденты:**

- **Создать аккаунт** — регистрация в системе для использования приложения.
- **Найти аудиокнигу** — поиск нужной аудиокниги в каталоге по названию или автору с вожможностью добавления кники в раздел "Послушать позже".
- **Слушать аудиокнигу** — начало прослушивания выбранной аудиокниги с вожможностьюю выбора диктора.
- **Загрузить аудиокнигу для прослушивания**:
  - загрузка аудиокниги из каталога для офлайн-прослушивания;
  - загрузка в приложение собственного файла из устройства.
- **Посмотреть список рекомендованных книг** — получение рекомендаций на основе выбранной книги по жанру.
- **Оставить оценку на прослушанную книгу** — постановка рейтинга (от 1 до 5) для прослушанной аудиокниги.
- **Добавить книгу в раздел «понравившиеся»** — добавление книги в закладки для последующего прослушивания.
