# Сущности

## 1. Аккаунт
**Описание:** Лицо, зарегистрированное в системе, может быть пользователем либо администратором.

**Атрибуты:**
- **Роль** — Администратор либо Пользователь
- **Логин** — Уникальное имя пользователя для входа
- **Пароль** — Набор символов, с помощью которого осуществляется вход в систему

---

## 2. Аудиокнига
**Описание:** Сущность, которая содержит информацию об аудиокниге.

**Атрибуты:**
- **Название** — Название книги
- **Автор** — Автор книги
- **Описание** — Краткое описание сюжета книги
- **Жанр** — Жанр книги
- **Изображение** — Обложка книги
- **Кол-во оценок** — количество оценок, поставленных пользователями
- **Имя диктора** — Имя рассказчика, читающего книгу

---

## 3. Администратор
**Описание:** Пользователь с правами администрирования, наследуемый от аккаунта.

---

## 4. Пользователь
**Описание:** Лицо, использующее приложение на повседневной основе. Не имеет дополнительных прав. Наследуется от аккаунта.

---

# Связи

- **Аккаунт — Администратор:** 1 к 1.  
  *Один аккаунт может быть привязан только к одному администратору.*

- **Аккаунт — Пользователь:** 1 к 1.  
  *Один аккаунт может быть привязан только к одному пользователю.*

- **Пользователь — Оценка:** 1 к 1.  
  *Один пользователь может выставить только одну оценку для каждой аудиокниги.*

- **Пользователь — Аудиокнига:** 1 к N.  
  *Один пользователь может получать доступ сразу к множеству книг.*

- **Администратор — Аудиокнига:** 1 к N.  
  *Один администратор может изменять (удалять, добавлять и редактировать) множество аудиокниг.*

- **Аудиокнига — Оценка:** 1 к N.  
  *Одна аудиокнига может иметь множество оценок от разных пользователей.*
